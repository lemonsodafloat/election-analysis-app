# 選挙分析アプリ
このアプリは、**小選挙区ごとに比例代表票を加算し、投票行動を可視化・分析**できるツールです。  
比例票の配分を自由に調整することで、「もし比例票がこう動いたら？」というシミュレーションを直感的に行えます。

## アプリURL
https://lemonsodafloat.github.io/election-analysis-app/

## 主な機能
- 小選挙区ごとの投票結果を可視化
- 比例代表票を自由に割り振って分析
- グラフによるリアルタイム反映

## 使い方
1. まず、分析したい**選挙区を選択**します  
2. 画面横の**サイドバーから比例票を自由に割り振り**ます  
3. 割り振った内容が、**中央のグラフにリアルタイムで反映**されます  
4. 表示結果は、**OS標準のスクリーンショット機能**を使って保存・共有してください ✨

## 動作環境
